Home Alarm System

Designed and constructed an entry system with keyless entry, intruder alarms, and other functionalities. Involved 3D printing various parts, wiring up a circuit that was compatible with an Arduino UNO, and writing code in Arduino. Tied together various aspects of event driven programming, electrical design, and mechanical engineering.

  • Date: Spring 2017
  • Field: Integrated Systems

Cigar Box Violin Design

In my ENGR 1620 course, we were given the task of developing a cigar box into a functioning violin. We created a model of the box using CAD and used software to determine the ideal frequency of our instrument. Afterwards we determined the dimensions of our instrument and built the violin.

  • Date: Fall 2015
  • Service: System Design

Bridge Investment Project

For my SYS 2001 class, my team and I were given a the list of over 13,000 highway bridges in Virginia and given the task of selecting which ones to repair with a give budget over a five year span. This required us to develop a series of objectives and measurable attributes for each bridge. Furthermore we made an executive recommendation and ran various types of sensitivity analysis on the data of the bridges we selected.

  • Date: Fall 2016
  • Service: Systems Engineering, Data Analysis

Delivery Robot

Designed and tested an autonomous robot with line following cabilities, allowing it to navigate to various destinations within a given map. Implements PID control and also has functionality to maintain a distance behind a "lead" robot. Uses Djikstra's shortest path algorithm to determine the quickest route to a given destination.
